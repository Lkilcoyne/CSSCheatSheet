CSS Selectors Reference

CSS Selectors

Tag Name Description
p Type Matches the type of "element" in the html Doc
* Universal Selects all elements
.'class attribute' Class Giving a description to an ellement. A way to select specific elements in CSS
#idname ID Giving an element an id attribute
[attribute] Attribute Used to target html elements that already contain elements
, Grouping Selectors Selects all matching selectors

Combinators

Tag Name Description
" " Descendant Combinator The space combinator selects elements that are descendants of the first element
> Child Combinator Selects elements that are direct children of the first element
~ General Sibling Selects siblings. The second element selects the first but not necessarily immediately. Both share the same parent
+ Adjacent Sibling Matches the second element only if it immediately follows the first
|| Column Combinator Selects elements that belong to a column
